基本命令

数据类型

String 类型

String 除了字符串类型还能是数字类型，一般用于

  • 计数器
  • 统计多单位数量
  • 粉丝数
  • 对象缓存存储

List 类型（列表）

Set 类型（集合）

Hash 类型（map）

本质上和String类型没有什么区别

Zset 类型（有序集合）

set的基础上增加了值

geospatial（地理位置）

geo的底层实现原理其实就是Zset
可以使用zset命令操作geo

hyperloglog（基数统计算法）

什么是基数
A{1,3,5,7,8,7,8}
B{1,3,5,7,8,8}
基数（不重复的元素的个数）=5，可以接受误差

bitmaps

位存储
一般用于存储0和1两种状态
